# v19: звук входящих сообщений и защита от случайных reload

Добавлено:
- звук входящего сообщения через Web Audio API;
- короткий звук отправленного сообщения;
- кнопка переключения звука сообщений;
- состояние звука сохраняется в `localStorage`;
- защита от случайных reload из-за кнопок без `type="button"`;
- динамические кнопки в модальных окнах нормализуются через JS;
- формы, которые не должны отправляться браузером, перехватываются.

Почему могли происходить reload:
- HTML-кнопка внутри формы без `type` по умолчанию становится `submit`;
- если обработчик не успел навеситься или упал, браузер отправляет форму и страница перезагружается;
- динамические кнопки в модальных окнах могли вести себя как submit.
