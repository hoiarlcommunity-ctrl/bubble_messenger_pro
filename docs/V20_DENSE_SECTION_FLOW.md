# v20: плотный поток элементов под разделами

Что исправлено:
- список чатов больше не растягивает элементы по всей высоте контейнера;
- раздел и его элементы теперь идут подряд:
  `Заголовок раздела -> элементы раздела -> следующий раздел`;
- минимизированы вертикальные отступы между разделом и первым элементом;
- минимизированы отступы между самими элементами;
- на мобильной версии применена та же логика.

Технически:
- контейнер `.dialogs` переведён из grid в плотный flex-column;
- отключено растягивание контента по высоте;
- добавлены жёсткие `margin/padding` override для section/item.
