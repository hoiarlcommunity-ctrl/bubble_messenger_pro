# v16: фикс бесконечного соединения в звонке

Что добавлено:
- таймаут WebRTC-соединения 18 секунд;
- понятный статус ICE/PeerConnection;
- если соединение не установлено, показывается ошибка вместо бесконечного `Соединение...`;
- кнопка `Повторить соединение`;
- предупреждение в консоль, если TURN-сервер не настроен;
- очистка таймера при завершении звонка.

Важно:
На Render сам Web Service не заменяет TURN. Для стабильных звонков между разными сетями нужен TURN-сервер:
`WEBRTC_TURN_URLS`, `TURN_USERNAME`, `TURN_CREDENTIAL`.
Без TURN звонки могут работать только в части сетей.
