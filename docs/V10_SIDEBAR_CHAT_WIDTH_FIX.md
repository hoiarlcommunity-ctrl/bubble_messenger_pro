# v10: Исправление ширины списка чатов

Проблема:
- карточки чатов в левом меню могли растягиваться по длинному тексту;
- preview/названия уходили под основной блок переписки;
- не хватало `min-width: 0` и `overflow: hidden` в grid/flex-элементах.

Исправлено:
- ограничена ширина левой колонки;
- карточки чатов теперь `width: 100%` и `max-width: 100%`;
- текст названия и preview обрезается через ellipsis;
- основной chat area защищён от горизонтального переполнения;
- добавлены mobile-ограничения для ширины.
