# v18: звуковое сопровождение звонков

Добавлено:
- исходящий рингтон при звонке;
- входящий рингтон при входящем звонке;
- короткий звук успешного соединения;
- короткий звук завершения звонка;
- короткий звук ошибки соединения;
- кнопка `Звук: вкл/выкл` в окне звонка;
- сохранение настройки звука в `localStorage`.

Реализация:
- звуки генерируются через Web Audio API;
- внешние mp3/wav-файлы не нужны;
- autoplay входящего рингтона может быть заблокирован браузером, если пользователь ещё ни разу не взаимодействовал со страницей.

Важно:
Для стабильных звонков онлайн всё равно нужен TURN-сервер. Звук не решает сетевую проблему WebRTC.
