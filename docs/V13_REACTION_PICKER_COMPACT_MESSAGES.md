# v13: компактные сообщения и встроенный выбор реакции

Что изменено:
- панель действий сообщения (`msg-tools`) больше не занимает место под сообщением;
- теперь это плавающая компактная панель поверх сообщения;
- убрано лишнее пустое пространство под сообщениями;
- кнопка реакции открывает компактный inline-picker возле сообщения;
- для чужих сообщений пикер появляется под сообщением;
- для своих сообщений пикер появляется над сообщением;
- реакция выбирается одним кликом, без `prompt()`.

Это уменьшает визуальную высоту сообщения и делает выбор реакции быстрее.
